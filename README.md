# pure-skill

## 简介 / Introduction

**中文**

pure-skill 用于根据项目代码结构，提纯并生成整体项目业务逻辑，留存完整的业务架构与数据架构。它会忽略具体技术栈、敏感数据与专有名词，输出一份可在其它项目中直接复用的、技术无关的业务逻辑文档。当用户要求「净化项目」「提纯项目」或输入 `pure` 时调用本技能。

**English**

pure-skill distills a project's overall business logic from its code structure and preserves the complete business and data architecture. It strips out the concrete tech stack, sensitive data, and proprietary terms, producing a tech-agnostic business-logic document that can be reused directly in other projects. Invoke this skill when the user asks to "purify" or "distill" a project, or types `pure`.
